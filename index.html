指标名称 指标计算公式 指标理想范围 指标含义描述
流动比率 流动资产 / 流动负债 大于1 衡量企业偿债能力，数值越大越好
速动比率 (流动资产 - 存货) / 流动负债 大于1 衡量企业短期偿债能力，数值越大越好
资产负债率 负债合计 / 总资产 小于1 衡量企业财务风险，数值越小越好
应收账款周转率 营业收入 / 应收帐款 高于行业平均值 衡量企业应收账款的收回速度，数值越大越好
存货周转率 营业成本 / 存货 高于行业平均值 衡量企业存货的周转速度，数值越大越好
存货周转天数 360 / 存货周转率 低于行业平均值 衡量企业存货的周转速度，数值越小越好
应付帐款周转率 营业成本 / 应付账款 高于行业平均值 衡量企业应付账款的支付速度，数值越大越好
应付账款周转天数 360 / 应付账款周转率 低于行业平均值 衡量企业应付账款的支付速度，数值越小越好
经营资金周转周期 存货周转天数 + 应收账款周转天数 低于行业平均值 衡量企业经营资金的周转速度，数值越小越好
经营资金垫付周期 营业周期 - 应付账款周转天数 低于行业平均值 衡量企业经营资金的垫付时间，数值越小越好
销售毛利率 销售毛利 / 营业收入 高于行业平均值 衡量企业销售活动的盈利能力，数值越大越好
销售净利率 净利润 / 营业收入 高于行业平均值 衡量企业销售活动的盈利能力，数值越大越好
总资产周转率 营业收入 / 总资产平均余额 高于行业平均值 衡量企业总资产的利用效率，数值越大越好
ROA 净利润 / 总资产 高于行业平均值 衡量企业利润与总资产的关系，数值越大越好
资产负债率 资产总额 / 股东权益总额 小于1 衡量企业财务风险，数值越小越好
净资产收益率（ROE） 净利润 / 权益平均余额 高于行业平均值 衡量企业净资产的盈利能力，数值越大越好
